\section{¿Qué es \LaTeX ?}
\begin{frame}
  Según \textcite{LaTeXWik47:online}, {\LaTeX} es:
  \begin{itemize}
    \item Sistema de composición de textos orientado a la creación de documentos escritos que presenten una alta calidad tipográfica.
    \item Por sus características y posibilidades, se usa de forma especialmente intensa en:
          \begin{itemize}
            \item Generación de artículos.
            \item Libros científicos.
            \item Expresiones matemáticas.
          \end{itemize}
  \end{itemize}
\end{frame}
\subsection{Ventajas de usar {\LaTeX}}

\begin{frame}[allowframebreaks]
  \begin{itemize}
    \item Separar diseño, del contenido. P.ej.:
    \begin{itemize}
      \item Numeración de referencias internas.
      \item Separación silábica.
      \item Creación de pies de página y bibliografías.
      \item Posicionamiento de flotantes.\footnote{Imágenes, tablas, diagramas, etc.}
      \item Generación de índices y tablas de figuras. Incluso de marcadores en el PDF final.
    \end{itemize}
    \item Uso de código abierto. El software no va a desaparecer o volverse de pago de la noche a la mañana.
    \item Existen opciones gráficas que facilitan su uso:
    \begin{itemize}
      \item TeXworks: Se integra muy bien con el entorno de desarrollo en Windows.
      \item LyX: Editor WYSIWYG.\footnote{Lo que ves, es lo que hay. Del Inglés \textit{What You See, Is What You Get}.}
      \item Overleaf: Editor web WYSIWYG con opciones de edición colaborativa. Tiene opción gratuita y descuentos para estudiantes.
    \end{itemize}
  \end{itemize}
\end{frame}

\subsection{Exámenes}
\begin{frame}
  Usando la clase \textit{exam}, podemos crear exámenes.

  \begin{itemize}
    \item LaTeX es ideal para escribir fórmulas matemáticas, e incluir distintos diagramas.
    \item Capacidad de crear espaciados entre preguntas.
    \item Posibilidad de crear soluciones a los exámenes. El espacio usado en la solución, se puede usar para crear el espacio de respuesta.
    \item Facilidad de crear exámenes tipo test o de opción múltiple.
    \item Posibilidad de añadir puntos a las preguntas (y sus partes), e incluso crear una rúbrica que poder añadir al final del examen.
    \item Con unos ajustes, se puede usar en castellano.
  \end{itemize}
  Ejemplo: \href{https://www.overleaf.com/learn/latex/Typesetting_exams_in_LaTeX\#Other_languages}{Overleaf: Typesetting exams in LaTeX}
\end{frame}

\begin{frame}
